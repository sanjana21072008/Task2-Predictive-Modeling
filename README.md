# Predictive Modeling Using Machine Learning

This project demonstrates a simple Machine Learning model using Linear Regression.

## Objective

To predict student marks based on study hours.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Steps Performed

1. Created a sample dataset.
2. Split the data into training and testing sets.
3. Trained a Linear Regression model.
4. Predicted marks for new input values.
5. Visualized the results using a graph.

## Outcome

Learned the basics of supervised learning and predictive modeling using Linear Regression.
